    About this app

      • There is a very unique phenomena found back in 2011 where if you pressed the first link of any Wikipedia page and do the same for the next page again, and again, you will eventually lead to the one same page, “Philosophy”. Just for the sake of curiosity, I created this app to help you to track the process of getting from any page into the ultimate page “Philosophy”. The only rules are:
        1. Only the main paragraphs
        2. Should not be other languages, sound, red, and external links
        3. Should not be the link inside of the brackets or parenthesis
